**Note1**: 关于`True or 1 / 0 or False`为什么是True而不是一个Error?
在 Python 中，布尔运算符 or 是一个短路运算符。当使用 or 运算符时，Python 会从左到右逐个计算每个表达式，一旦找到一个为 True 的表达式，就会立即返回该值，而不再继续计算后面的表达式。
